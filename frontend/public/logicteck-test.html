Standalone page — no ICS login. Look up ICS transfer data by reference (e.g. ICS-202600018) to receive pre-forecast details and book the empty return on LOGICTECK via the public Railway API.

OUTSIDE ICS APP · PUBLIC API ONLY
Permanent connection: Every ICS transfer includes a transferLink block with the reference, trucker name, and API URLs. LOGICTECK uses the reference to pull the dossier and create the empty-return booking. ICS also auto-pushes data when the transfer QR is published (if Logicteck:BookUrl is set). No shared user accounts or JWT — only the API key and ICS reference.
