# Intelligent Container Solutions (ICS)

## Short description

ICS streamlines empty container returns from pre-forecast to gate pass — connecting truckers, shipping lines, and depots on one platform.

## Full description

**Intelligent Container Solutions (ICS)** is a container return management platform that digitizes the full empty-container lifecycle: from trucker pre-forecast submission through shipping-line evaluation, container yard allocation, depot scheduling, payment verification, and LOGICTECK QR booking.

ICS replaces scattered spreadsheets, phone calls, and paper workflows with a single, auditable portal shared by truckers, shipping-line evaluators, depot personnel, and administrators. Whether a container is returning to a **Container Yard (CY)** or a **Port Terminal**, every step stays visible, tracked, and coordinated in one place.

Truckers can self-register to submit pre-forecasts (with required container photos), track assigned returns, upload payment proof, manage withdrawals (ATW), handle demurrage billing, and download booking QR codes — on both the web portal and the ICS Trucker mobile app. Shipping-line evaluators approve or request compliance corrections with full dossier visibility. Depot teams allocate yard capacity, assign return slots, confirm truckers, and release containers with real-time status updates.

ICS does not replace each company’s internal operational systems — it provides the shared communication, validation, and audit layer that keeps every party aligned from first submission to gate release.

## Key features

- **Pre-forecast & photos** — Truckers submit container details with required identity photos before shipping-line review begins.
- **Shipping-line evaluation** — Evaluators approve or request compliance corrections with full dossier visibility and audit history.
- **CY allocation & inventory** — Track yard capacity, manual inventory, releases, and TEU limits across depots and shipping lines.
- **Depot scheduling** — Assign return slots, confirm truckers, and coordinate gate activity with real-time status updates.
- **Payments & demurrage** — Upload payment proof (with OCR assist), manage return fees, and handle demurrage billing in one place.
- **Authority to Withdraw (ATW)** — Digitize withdrawal requests for container repositioning, from ATW issue through yard validation and release.
- **LOGICTECK QR booking** — Generate and print booking QR codes, sync status with LOGICTECK, and keep truckers informed.
- **Role-based access** — Dedicated workflows for truckers, evaluators, depot personnel, and administrators.
- **Push & in-app notifications** — Alerts for schedule, payment, withdrawal, and news updates on web and mobile.
- **Trucker mobile app** — Native Android app for pre-forecast, returns, withdrawals, payments, QR passes, and profile management on the go.
- **Complete audit trails** — Maintain a clear, auditable path from trucker submission to depot release.
